# asr_whisper
## 基于whisper的语音识别

# 环境
python == 3.10  
使用命令行配置环境  
```
pip install -r requirments.txt
```
whisper的安装详见|https://github.com/openai/whisper.git  
在初次使用时，会下载whisper的模型，运行时间可能会较长。
下一次运行则不会再次下载模型，除非更换未使用过的模型。

# 使用
使用命令行直接运行
```
python ui.py
```
